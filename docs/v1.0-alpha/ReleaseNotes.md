# v1.0-alpha Release Notes

## Added

- Introduced the new `Life Buddy` screen as a rule-based in-app assistant.
- Added Life Buddy access from the home dashboard and desktop sidebar navigation.
- Added local suggestions for today's priorities, weak categories, overdue relationship to-dos, pending KPI entries, and people needing attention.

## Changed

- Renamed all user-facing `Copilot` references to `Life Buddy`.
- Renamed the route from `/copilot` to `/life-buddy` while preserving the same functionality and rule-based logic.
